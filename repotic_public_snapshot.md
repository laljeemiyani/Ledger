# LedgerFlow Public Feature & Policy Snapshot (For CA Review)

## 1. Executive Summary
Repotic is a web-based tool aimed at e-commerce sellers and Chartered Accountants. It focuses on converting bank statements and marketplace reports (PDF/CSV) into structured accounting outputs like Tally XML, Excel, and CSV. Its publicly visible features highlight automation for GST reporting, e-commerce reconciliation, and statement processing.

## 2. Key Public Features
- Statement conversion from PDF bank statements to Tally XML, Excel, or CSV formats.
- Uploading and processing of marketplace sales data (Amazon, Flipkart, Meesho, PayTM, etc.).
- GST and GSTR-1 reporting assistance based on automated data extraction.
- Demos and tutorial videos to showcase workflows.
- User signup, login, and subscription-based paid plans.

## 3. Inputs & Outputs
**Inputs Supported (from public claims):**
- PDF bank statements
- CSV statements
- Marketplace-generated CSV or report files

**Outputs Supported:**
